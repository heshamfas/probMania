Changing game options

Picture shows the options dialog box which allow you to choose 
your preferred setting 

       The option dialog box  gives the PropGame the real taste, once you feel comfortable with the game you can  increase its complexity to provide you with more excitements and experience.

  •        Maximum bet: This option will set the figure that the bet should not go above. The generated bets will never go higher than the chosen option.
  •        Minimum bet: This option will set the  figure that the bet should not go below. The generated bets will never go lower than the chosen option.
  •       Increments: This Option will set  the how much the bet should be incremented to make the bet more realistic, For Example if you choose increments of $5 the the bet will go $5 $10 $15 etc.... that will eliminate struggling with silly bets like $27 any craps or $189 C&E. That way you can save the energy and time to practice bets more likely to be seen on the table. The lower the increments setting with high maximum bet setting  the more the complicated the game will be.
  • Number of players: this option will set the highest number of players to deal to, You might see a lower number of players than the set number with each different rolls but you'll never see a higher than the option set.
      

  • Note: 

  •        The Minimum bet amount must be lower than the Maximum bet amount.
  •       Also the increments must be lower than the minimum bet amount except if they are both set to $1.
  •       To see the ugliest bets you can ever imagine set the Maximum bet to $5000,Minimum bet to $500 and increments to $1
