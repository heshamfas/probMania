Getting Started 

System requirements:

  • Windows 95,98,NT,2000 or higher.

  • Intel 100 MHz processor or higher.

  • 32 MB of RAM (64 recommended).

  • 10 MB of hard disk space (50 MB is recommended).

  • 256  color display or higher (16 Millions colors is recommended)

  • CD ROM drive.

 

 

 

 

Installing the game :

  1.  Insert PropMania CD into your CD drive

  2. A pop up installation dialog box will appear 

  3.  Follow Instructions clicking the next button.

  4.  If you have 50 MB of hard disk space available on your hard disk then select all the available  components for better performance

  5. Click Finish to end the installation process.

 

 

 

 

 

Starting the game

  • If the installation process ended successfully there should be an icon appear on the Desktop called PropMania, double click it.

  • If you chose Not to install this Shortcut on your Desktop then click the start menu button

  • Scroll up to programs then scroll down to HeshamFas Games.

  • Scroll right to PropMania then click on PropMania.exe Shortcut.

  • The Game Should be starting now

  • If you didn't choose full installation then you'll need to insert your PropMania CD into your CD ROM drive

  • Have fun.
