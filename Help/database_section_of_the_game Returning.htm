Returning user

The Picture above shows a returning user clicked on his name and his history is shown in the history panel 
 
  • Click on your name or the name you have previously assigned for yourself.
  • Use the Up and Down buttons to move the name above or below others
  • If you want to remove a user name permanently  from the list click the remove button.
  • A dialog box will appear warning you that the name will be removed permanently.
  • Click yes if you sure or no to cancel
  • Click the lets go button to play the PropGame.
